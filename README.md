# Trabalho_Pratico_web
Um website front-end e back-end completos, funcionais e visualmente consistentes, inclui uma ligação a base de dado de modo a aceder às 3 tabelas necessárias para o seu funcionamento.

Nome do projeto: 
Inês Cruz - Tradutora Freelance. 

Tema escolhido: 
Website de tradutora freelancer para apresentação da mesma a prospetivos clientes. 

Tecnologias utilizadas: 
html, css, bootstap, javascript, php, composer, phpmailer. 

Funcionalidades principais: 
front-end:
página de apresentação, 
portfolio á base de tabs, 
grupo co-criado pela tradutora apresentado atraves de acordião, 
informaçâo sobre a tradutora (apenas containers), 
contacto com a tradutora por links ou envio de email através de formulário (com verificação).

back-end:
ligação a base de dados,
atualização da base de dados,
simulação de orçamentos conforme os dados das tabelas da base de dados,
visualização de histórico pelos clientes,
envio opcional da simulação ao adminstrador por phpmailer,
área do administrador,
visualização do histórico de todos os clientes ou apenas de um especifico,
eliminar clientes juntamente com o seu histórico.


Eventuais limitações ou ideias futuras: 
Passar o email da página contacos para phpmailer,
melhorias visuais,
versão em inglês do site
