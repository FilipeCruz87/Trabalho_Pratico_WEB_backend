# Trabalho_Pratico_web
Um website front-end e back-end completos, funcionais e visualmente consistentes, inclui uma ligação a base de dado de modo a aceder às 3 tabelas necessárioas para o seu funcionamento.

Nome do projeto: 
Inês Cruz - Tradutora Freelance. 

Tema escolhido: 
Website de tradutora freelancer para apresentação da mesma a prospetivos clientes. 

Tecnologias utilizadas: 
html, css, bootstap, javascript, php, composer. 

Funcionalidades principais: 
página de apresentação, 
portfolio á base de tabs, 
grupo co-criado pela tradutora apresentado atraves de acordião, 
informaçâo sobre a tradutora (apenas containers), 
contacto com a tradutora por links ou envio de email através de formulário (com verificação).
